# Portainer

## Purpose

Provides a web interface for managing Docker containers, networks, images and volumes.

## Why Portainer?

* Simplifies Docker management
* Easy container monitoring
* Visual management interface
* Useful learning tool

## Access

https://192.168.1.130:9443

Container URL:

https://portainer:9443

## Important Data

Docker Volume:

* portainer_data

Contains:

* Users
* Settings
* Endpoints

This volume should be backed up.
